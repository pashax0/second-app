# p014_admin — Admin Web Interface

## Goal

Отдельное веб-приложение для управления магазином: создание товаров и дропов, публикация витрины.

## Non-goals

- Управление заказами
- Push-уведомления о новом дропе
- Пометка товара как sold-out (через покупку в мобильном приложении)
- Редактирование/удаление дропов

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- TanStack Query
- React Hook Form + Zod
- Supabase JS (тот же проект, те же env vars)

## Бизнес-модель (зафиксировано)

- Одна активная **витрина** одновременно — текущий дроп с 15–20 товарами
- Дроп обновляется ежедневно: старые товары заменяются новыми
- Нераспроданные товары можно переиспользовать в следующем дропе (возможно с другой ценой)
- Статусы товара: `draft` → `available` → `reserved` (с таймером) → `sold`
- Корзина блокирует товар на N минут (по умолчанию 10, настраивается per-user)
