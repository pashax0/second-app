# UX Decisions

Зафиксированные решения — не менять без явного обсуждения.

## Главный экран

**Сетка фото 2-3 колонки (в зависимости от экрана)** — все товары текущего дропа, только фото (как Instagram-профиль).

- Опциональный хедер с описанием дропа (если задано)
- Sold-out товары остаются в сетке: полупрозрачные + оверлей ❌ (FOMO)
- Название дропа, дата, номер — пользователю не показываем

## Карточка товара

Полноэкранный просмотр с **горизонтальным свайпом** по всем товарам дропа.

- Фото большое, текст под ним: бренд, размер, обмеры, цена, кнопка
- Счётчик позиции в хедере: "3 / 9"
- Sold-out: кнопка "Купить" заменяется на "Продано"

## Архив

Список прошлых дропов. Тап → та же сетка, что и на главной. Нераспроданные вещи можно купить.

## Принципы

- Фото и описание товара **всегда вместе** — проблема инстаграма, от которой уходим
- Минимум шагов между "увидел" и "купил"
- Визуальность первична: сетка → тап → детали → покупка
