# Business Context

## Что это

Секонд-хенд магазин одежды. Вещи привезены преимущественно из Англии, США, Испании —
качественные западные бренды (Regatta, PLT, Zara, M&S, Columbia, River Island и т.д.).

Магазин работает через Instagram (@2day_wear). Приложение — переход на собственную платформу.

## Как работает

- **Витрина** — единственный экран для покупателей, содержит 15–20 товаров текущего дропа
- Обновляется ежедневно: все старые товары заменяются новыми
- Каждая вещь уникальна — одна в одном экземпляре
- Цена у каждой вещи своя
- Оплата при получении (наложенный платёж)
- Возврат возможен, если вещь не подошла (обмен не моделируется — см. [product-lifecycle.md](product-lifecycle.md))
- Архива прошлых дропов для покупателей нет — только текущая витрина

## Дроп (внутренняя сущность)

- Хранится в БД: название (обязательно), описание (опционально), дата публикации
- Связан с набором товаров через `drop_items`
- Нераспроданные товары из прошлого дропа не удаляются — админ может включить их в следующий (возможно с другой ценой)

## Жизненный цикл товара

Статусы, переходы, связанные сущности (партии, возвраты, списания), правила
soft-delete и что сознательно НЕ моделируем — в [product-lifecycle.md](product-lifecycle.md).
Здесь не дублируем.

## Резервация (корзина)

- Товар на витрине (`status=listed`) можно добавить в корзину — создаётся `reservations` row
- Другие пользователи видят: товар занят + сколько времени осталось до освобождения
- Таймер по умолчанию: 10 минут
- Настраивается из админки: глобально или per-user (новый → меньше, доверенный → больше)
- По истечении резервация удаляется (через client RPC `expire_reservation`, см. [ADR-004](../../docs/decisions/ADR-004-reservation-expiry.md)) — товар снова доступен. `products.status` остаётся `listed`.

## Целевая аудитория

Подписчики инстаграм-магазина. Привыкли листать сетку фото и писать в директ чтобы купить.
Приложение должно сохранить визуальный ритм инстаграма, убрав неудобства.

## Маркетинговая механика

Drop Marketing + Scarcity Marketing: единственный активный дроп, уникальные вещи, таймер корзины —
всё работает на создание ощущения дефицита и срочности. Цель: "решился — покупай сейчас".
